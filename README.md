# stm32_cMake_build_system
This Project shows a CMake build system working on a NucleoG431RB.

# Tools Needed
- install cmake (tested with 4.2.3)
- download: arm-gnu-toolchain (tested with Arm GNU Toolchain 13.3.Rel1 (Build arm-13.24)) 13.3.1 20240614)
- install ninja (tested with 1.13.2)
- CubeProgrammer (tested with v2.21.0)
